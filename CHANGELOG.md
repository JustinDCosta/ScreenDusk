# Changelog

## [3.0.0] - 2026-03-26

### Added
- Customizable global hotkeys with persistent settings.
- System tray controls and startup integration.
- Publish script support for local and PFX-based code signing.
- PerMonitorV2 high-DPI configuration for sharper UI rendering.
- GitHub release workflow and repository security/release documentation.

### Changed
- Polished main window layout and title bar controls.
- Strengthened project metadata and versioning for release management.
- Refined build/publish process for repeatable distribution.

### Fixed
- Multi-build output confusion by standardizing release output guidance.
- UI clipping issues by making central content scrollable.
